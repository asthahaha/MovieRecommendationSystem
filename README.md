# 🎬 Movie Recommendation System

## 📌 Overview

A Machine Learning based Movie Recommendation System that recommends similar movies based on user preferences using content-based filtering techniques.

---

## 🚀 Features

- Movie recommendation based on similarity
- Fast recommendation engine
- Uses cosine similarity
- Interactive prediction

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- Jupyter Notebook

---

## 📂 Dataset

movies.xlsx

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python astha.py
```
